# 🚀 Job Portal Web Application

A full-stack Job Portal application developed using **Spring Boot**, **React**, **MySQL**, and **JWT Authentication**. The platform provides separate functionalities for Users, Recruiters, and Admins with role-based access control.

---

## 📌 Features

### 👤 User Features

* Register & Login
* JWT Authentication
* Search Jobs
* Apply for Jobs
* View Applied Jobs
* Save Jobs
* View Saved Jobs
* Upload Resume
* Download Resume
* Edit Profile
* Change Password

---

### 🏢 Recruiter Features

* Recruiter Dashboard
* Create Job
* Edit Job
* Delete Job
* Manage Posted Jobs
* View Applicants
* Download Applicant Resumes
* Accept/Reject Applications
* Email Notifications

---

### 🛠 Admin Features

* Admin Dashboard
* Statistics Cards
* Manage Users
* Manage Jobs
* Delete Users
* Delete Jobs

---

### 🔍 Search and Filters

* Search by Job Title
* Filter by Company
* Filter by Location
* Filter by Job Type
